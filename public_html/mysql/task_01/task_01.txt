Задание 1. Создание и модифицирование  простейших таблиц (выполнить до ????-??-??_ 23:59)

Аналогично описанному в 
https://dev.mysql.com/doc/refman/5.6/en/creating-tables.html
создать таблицу, изменить содержание в таблице и получить данные из таблицы с использование запросов типа
  
000_Creating_a_Table - Создать таблицу и добавить два поля различного типа
010_Inserting_Into_Table - Дополнительно к примеру добавить две записи с несколькими значениями по умолчанию
015_Selecting_All_Data  - Отобразить содержание всей таблицы   - не выполнять - использовали в 010_...
025_Selecting_ Particular_Rows  - Отобразить содержание  четырех полей с названиями колонок
025_Selecting_ Particular_Columns -new- Отобразить содержание  четырех полей с оригинальными названиями колонок   
030_Selecting_Particular_Columns - Привести  два оригинальных примера  получения данных из таблицы по условию
030_Selecting_Particular_Where -new- Привести  два оригинальных примера  получения данных из таблицы по условию
035_Sorting_ Rows  - Выполнить запросы сортировок по дате рождения, продолжительности жизни, по полу 
040_Data_ Calculations _ Привести примеры расчетов с использованием содержания таблицы
045_Pattern Matching - Выполнить два поиска  данных в таблицах с использованием шаблонов
051_Counting_Rows - привести два примера агрегированных запросов
052_Counting_Rows - привести два примера запросов с агрегированием и сортировкой  
055_ALTER_ADD_AUTO_INCREMENT_PRIMARY   - добавить первичный ключ

Итого 10 пунктов (015 не выполнять)


В названиях всех полей добавить префиксы, соответствующие  логину.  (см пример 000_Creating_a_Table.*)
